# Мои проекты

См. также: [Dynamic Pricing System for Rental Properties](https://github.com/Timkavipww/projects/blob/main/dynamic_price_project.md)

Краткое описание:
Разработал систему динамического ценообразования для аренды недвижимости: автоматический сбор рыночных данных, анализ спроса и цен конкурентов, расчёт рекомендованной цены в зависимости от горизонта бронирования и состояния рынка. Реализовал парсинг динамических страниц, агрегацию данных, Telegram-уведомления и автоматизированный пайплайн обработки данных.
