# Проект: Event Management Platform

#### Суть

Разработал платформу для управления мероприятиями с интеграцией Telegram Mini App и Telegram-бота.
Система позволяет администраторам создавать и управлять мероприятиями через административную панель, а пользователям — просматривать события, регистрироваться на них и получать уведомления через Telegram.

#### Проблема

Организация мероприятий вручную создает ряд проблем:

* отсутствие централизованной системы регистрации участников
* сложность сбора и хранения данных пользователей
* неудобное управление мероприятиями и списками участников
* необходимость синхронизации веб-интерфейса и Telegram
* проблемы с обработкой времени и часовых поясов

Из-за этого организаторы тратят много времени на обработку заявок и поддержку актуальной информации.

#### Реализация

* Разработал backend API для работы Telegram Mini App и административной панели.
* Реализовал Telegram-бота для взаимодействия с пользователями и отправки уведомлений.
* Настроил JWT-аутентификацию через Telegram InitData.
* Реализовал систему регистрации пользователей на мероприятия с защитой от повторной записи.
* Построил архитектуру на основе FastAPI, PostgreSQL и Docker Compose.
* Реализовал хранение и обработку дат в UTC с автоматическим преобразованием времени в МСК для отображения пользователям.
* Настроил работу с PostgreSQL/PostGIS и геоданными через GeoAlchemy2.
* Реализовал асинхронную работу backend-сервисов и базы данных через asyncpg и SQLAlchemy 2.0.
* Добавил обработку загрузки файлов и экспорт данных в Excel.
* Настроил контейнеризацию сервисов: backend, frontend, Telegram-бот и PostgreSQL.
* Реализовал систему логирования, healthcheck и автоматического перезапуска контейнеров.

#### Ключевые возможности системы

* Создание и управление мероприятиями через админ-панель
* Регистрация пользователей на мероприятия
* Telegram Mini App для работы пользователей
* JWT-аутентификация через Telegram
* Хранение данных участников и истории регистраций
* Защита от повторной записи на одно мероприятие
* Экспорт данных участников
* Работа с геоданными и координатами
* Асинхронная обработка запросов
* Контейнеризация и orchestration через Docker Compose
