### Проект: Dynamic Pricing System для аренды недвижимости

#### Суть

Разработал систему динамического ценообразования для краткосрочной аренды недвижимости на основе рыночных данных с платформы Ostrovok.
Система ежедневно собирает и анализирует цены конкурентов, динамику спроса и изменение количества доступных объектов, после чего формирует рекомендации по оптимальной цене размещения.

#### Проблема

Ручное управление ценами в аренде недвижимости не учитывает:

* изменение спроса по мере приближения даты заезда
* колебания рыночных цен
* сокращение количества доступных объектов
* сезонность и поведение рынка

Из-за этого собственники либо теряют прибыль, либо получают простой объекта.

#### Реализация

* Разработал пайплайн автоматического сбора данных по ценам и доступности объектов аренды.
* Реализовал парсер динамических страниц с использованием браузерной автоматизации и HTML-парсинга.
* Построил систему анализа рынка на основе:

  * horizon pricing (горизонт до заезда)
  * медианных цен
  * сегментации рынка
  * оценки спроса через изменение количества предложений
* Реализовал алгоритм расчёта рекомендованной цены в зависимости от:

  * времени до заезда
  * рыночной позиции
  * динамики спроса
  * ценового сегмента
* Настроил автоматический запуск задач, логирование и обработку ошибок.
* Реализовал Telegram-бота для уведомлений и мониторинга работы системы.
* Подготовил основу для дальнейшего перехода к автоматическому pricing engine и ML-моделям прогнозирования.

#### Ключевые возможности системы

* Ежедневный сбор и агрегация рыночных данных
* Анализ изменения цен по горизонтам бронирования
* Вычисление median / p25 / p75 для оценки рынка
* Определение позиции объекта относительно конкурентов
* Выявление роста и падения спроса
* Генерация рекомендаций:

  * повысить цену
  * снизить цену
  * удерживать текущую цену
