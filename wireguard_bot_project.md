# Проект: WireGuard Configs Manager

#### Суть

Разработал систему управления VPN-конфигурациями WireGuard с интеграцией Telegram-бота и автоматизированной системой оплаты через YooKassa.
Пользователи могут приобрести VPN-конфиг, получить инструкции по подключению, управлять своей подпиской и активировать пробный доступ напрямую через Telegram.

#### Проблема

Ручная выдача VPN-конфигураций создает ряд проблем:

* необходимость вручную проверять успешность оплаты
* ручная выдача и продление конфигов
* отсутствие автоматического контроля срока действия подписок
* сложность управления большим количеством пользователей
* высокая нагрузка на администратора при обработке запросов

При росте количества пользователей такой подход становился неудобным и плохо масштабировался.

#### Реализация

* Разработал Telegram-бота для покупки и управления WireGuard-конфигами.
* Реализовал интеграцию с YooKassa для автоматической обработки платежей.
* Настроил асинхронную обработку платежей через Celery и Redis.
* Реализовал отдельные background tasks для проверки статуса платежей и автоматической выдачи конфигов.
* Построил систему автоматической проверки просроченных конфигураций и деактивации доступа.
* Реализовал реферальную систему для пользователей.
* Добавил возможность получения пробного VPN-конфига.
* Реализовал управление конфигурациями через HTTP API.
* Добавил генерацию QR-кодов для быстрого подключения WireGuard.
* Реализовал отправку инструкций и гайдов по подключению VPN-клиента.
* Построил backend на FastAPI и SQLAlchemy с асинхронной работой через asyncpg.
* Настроил контейнеризацию и автоматический запуск сервисов.

#### Ключевые возможности системы

* Покупка VPN-конфигов через Telegram
* Интеграция с YooKassa
* Автоматическая выдача и продление конфигураций
* Проверка и обработка платежей в background tasks
* Реферальная система
* Пробный VPN-доступ
* Генерация QR-кодов для подключения
* Автоматическое отключение просроченных конфигов
* Асинхронная обработка задач через Celery
* Управление WireGuard-конфигами через API
